# Secure Enterprise Network Projects

This repository contains advanced network security and infrastructure projects developed using Cisco Packet Tracer.

## 1. Secure Company Network System

This project focuses on the implementation of a secure enterprise network featuring an ASA Firewall, L2/L3 switching, and redundant routing.

### Network Topology
![Network Topology](./1.Design%20and%20Implementation%20of%20a%20Secure%20Company%20Network%20System%20(Project%20%231)/Practical%20Screenshots/Network%20Topology.png)

### Key Configurations
- **ASA Firewall**: Security zones, levels, and inspection policies.
- **Switching**: VLANs, EtherChannel, and STP (Portfast/BPDU Guard).
- **Routing**: OSPF for dynamic routing and HSRP for gateway redundancy.
- **Services**: DHCP, Static IP for servers, SSH management, and WLAN.
- **Advanced**: VoIP configuration for business communication.

### Project Files
- **Packet Tracer File**: `Secure Company Network System.pkt`
- **Configurations**: Detailed configs for Routers, Switches, and Firewalls in respective folders.
- **Addressing Plan**: Documentation for the IP addressing scheme.
